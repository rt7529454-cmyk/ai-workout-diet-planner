# Personalised Workout & Diet Planner with AI

This project generates personalized workout and diet plans 
based on user fitness data using AI concepts.

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- NumPy
- Pandas

## Features
- BMI Calculation
- Goal Classification
- Workout Recommendation
- Diet Suggestion
